Software Takes Command
our contemporary society can be characterized as a software society and our culture can be justifably called a software culture—because today software plays a central role in shaping both the material elements and many of the immaterial structures that together make up “culture.
Critical code studies is an emerging feld related to software studies and platform studies, but it’s more closely attuned to the code itself of a program rather than the program’s interface and usability (as in software studies) or its underlying hardware (as in platform studies).
